## Mekky E-Commerce Data Analytics Project: Driving Data-Driven Growth

This project provides a comprehensive data analysis of Mekky's entire operational pipeline, moving the company from intuitive decision-making to a data-backed growth strategy. The analysis addresses core business challenges related to customer retention, operational efficiency, and profitability.

### Key Objectives & Business Questions Answered

The analysis was structured around five critical areas:
1.  **Customer Behavior & Churn Risk:** Segmenting customers to identify the most valuable buyers and predicting which customer groups are at the highest risk of attrition.
2.  **Operational Efficiency:** Pinpointing regions and factors responsible for the longest delivery delays.
3.  **Marketing ROI (Return on Investment):** Evaluating campaign performance to optimize spending.
4.  **Product Performance:** Identifying high-margin, top-selling product categories.
5.  **Customer Feedback:** Linking sentiment analysis directly to purchasing behavior (e.g., proving that unhappy customers churn faster).

### Technical Stack
* **Language:** Python (Pandas, NumPy)
* **Visualization:** Matplotlib, Seaborn, Plotly
* **Environment:** Jupyter Notebook / Lab / VS Code
